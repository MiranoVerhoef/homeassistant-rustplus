# Rust+ MQTT Bridge (Home Assistant Add-on)
Exposes Rust+ smart devices (switches, alarms, cameras) to Home Assistant using MQTT Discovery.

Features
MQTT Discovery (no custom integration needed)
Smart Switch control
Smart Alarm status
CCTV camera frames (if enabled server-side)
Installation
Add this GitHub repository to Settings → Add-ons → Add-on Store → Repositories
Install Rust+ MQTT Bridge
Configure MQTT + Rust+ credentials
Start the add-on
Notes
No admin access required
Requires Rust+ pairing credentials
