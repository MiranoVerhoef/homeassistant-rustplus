# Rust+ MQTT Bridge (Home Assistant Add-on)

Exposes Rust+ smart devices (switches, alarms, cameras) to Home Assistant
using MQTT Discovery.

## Features
- MQTT Discovery (no custom integration needed)
- Smart Switch control
- Smart Alarm status
- CCTV camera frames (if enabled server-side)

## Installation
1. Add this GitHub repository to **Settings → Add-ons → Add-on Store → Repositories**
2. Install **Rust+ MQTT Bridge**
3. Configure MQTT + Rust+ credentials
4. Start the add-on

## Notes
- No admin access required
- Requires Rust+ pairing credentials
